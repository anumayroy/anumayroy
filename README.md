Hi, I'm Anumay Roy, a passionate Game Developer and 3D Generalist from India.

I'm the founder of YugaLoop, an independent game studio focused on creating immersive games and interactive experiences. I specialize in Unreal Engine 5, Unity, Blender, Maya, ZBrush, Substance Painter, and AI-powered content creation workflows.

I enjoy building complete game experiences—from gameplay programming and environments to character creation, animation, UI, VFX, and optimization. I constantly explore new technologies including generative AI, procedural workflows, and automation to improve development speed and quality.
